Loop 'n Crash

Options:

  • Invoke an endless loop in the application by accessing /endless-loop
  • Invoke a crash in the application by accessing /crash
  • Invoke a response displaying the GET parameters by accessing /show-get-params
  • Invoke a response with a specific status code by accessing /status-code followed by a three-digit status code
  • Invoke a slow response by accessing /delay followed by a number representing the amount of seconds to delay.
