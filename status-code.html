Status Code

Options:

  • Invoke a 200 (OK) status: /status-code/200
  • Invoke a 301 (Moved Permanently) status: /status-code/301
  • Invoke a 402 (Paymeny Required) status: /status-code/402
  • Invoke a 503 (Service Unavailable) status: /status-code/503
  • Invoke any other status code by appending it's numeric value to the /status-code/ URL
